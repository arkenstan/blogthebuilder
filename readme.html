Welcome, To Blogthebuilder readme file. Here you can find answers to most of your queries

Installation (for Webserver)

  • Extract the Package
  • Create a Database with any name using your cPanel or Database
  • Open connect.php file in the extracted package(with your text editor)
  • Replace your database credentials in HOSTNAME, USERNAME, PASSWORD, DATABASE
  • Upload package to your public_html folder on webserver
  • Now open installation.php (yourwebsite.com/installation.php) and your are good to go.

Installation (for Localhost)

  • Extract the Package
  • Copy and Paste extracted package in your localhost folder (htdocs or www)
  • Create a Database with any name
  • Open connect.php file from the extracted package(with your text editor)
  • Replace your database credentials in HOSTNAME, USERNAME, PASSWORD, DATABASE
  • Now open installation.php with localhost and your are good to go.
