Installation

If you are reading this readme file means you have downloaded the Blog The Builder package and extracted it or uploaded it to your web server for installation please follow this simple step guide :

  • In extracted package you will find file connect.php please open it in an text editor and update credentials for HOSTNAME, USERNAME, PASSWORD, DATABASE according to your web server
  • Upload the modified file (connect.php) back to your web server
